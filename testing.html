Running the Tests¶

Just run:

$ python setup.py test

If you want you can run a single module, say the methods suite:

$ python setup.py test -s eve.tests.methods

Or, to run only the get tests:

$ python setup.py test -s eve.tests.methods.get

You can also choose to just run a single class:

$ python setup.py test -s eve.tests.methods.get.TestGetItem

Or even a single class function:

$ python setup.py test -s eve.tests.methods.get.TestGetItem.test_get_max_results

RateLimting and Redis¶

In order to run successfully the Rate Limiting suite needs both a running Redis server and redispy to be installed. If any of these conditions is not satisfied the suite will just be silently skipped.

While you have to install Redis by yourself (it is really easy), redispy will install automatically on the first test run. Of course you can still install it yourself, if so you wish:

$ pip install redis

You are looking at the documentation of the development version.
